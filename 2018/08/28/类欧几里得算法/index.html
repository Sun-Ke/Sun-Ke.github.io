在二维坐标系有两条直线y=abx,y=cdxy=\frac{a}{b}x,y=\frac{c}{d}xy=ba​x,y=dc​x。对于每一条直线，有一只蚂蚁从(1,0)出发，只能向上或向右走一格，并且蚂蚁一直都在直线下方，每次优先向上走，如果越过了直线就改为向右走。问对于这两个蚂蚁走过的点的交集大小。

斜率相等时交集点数无穷大。

对于直线y=abxy=\frac{a}{b}xy=ba​x，走到的点为ab(x−1)−1<y≤abx,x≥1\frac{a}{b}(x-1)-1 < y \le \frac{a}{b}x,x\ge 1ba​(x−1)−1<y≤ba​x,x≥1

假设直线y=cdxy=\frac{c}{d}xy=dc​x斜率较小，那么交集为

ab(x−1)−1<y≤cdx,1≤x≤n\frac{a}{b}(x-1)-1 < y \le \frac{c}{d}x, 1 \le x \le n ba​(x−1)−1<y≤dc​x,1≤x≤n

n=⌊d(a+b)ad−bc⌋n=\lfloor\frac{d(a+b)}{ad-bc}\rfloor n=⌊ad−bcd(a+b)​⌋
